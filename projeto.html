Historia da Internet

(voltar)

O primeiro computador foi criado em 1946 O famoso ENIAC que faz parte da primeira geração de computadores

juntamente com COLLOSSUS e MARK1 durante a guerra fria entre China e Estados unidos.

A finalidade desses computadores sempre foi calculos cientificos e na decada de 1950 foi criada a DARPA

(agencia de projetos e pesquisas avançadas).

Com o avanço das pesquisas comecou-se uma teoria de comunicação entre pessoas por uma rede iniciada por Joseph Licklider

então surgiu a ARPANET(pesquisas cientificas e militares) com conceitos de checkpoints entre seus lugares

Até que aconteceu a primeira conexão entre esses pontos que foi em 29/10/1969

Depois do surgimento dessas instituições especificas um homem chamado TIM BERNES-LEE criou um conceito de rede mundial

para que tudo ficasse mais organizado então surgiu a famosa INTERNET

TIM BERNERS-LEE é conecido como o pai da internet pois criou o (W.W.W)

e da primeira pagina web. Foi dessa maneira que surgiu a Internet que usamos até os dias de hoje

Conceito Cliente

(voltar)

A interface do cliente é a parte que aparece quando ele acessa um site de algum lugar atraves da internet

essa informação normalmente vem de um servidor e esse servidor armazena essa informação no banco de dados

quando a informação é solicitada pelo usuario a informação é buscada atraves do id protocolo HTTP

as ferramentas utilizadas nesse lado (front-end) são:

HTML

CSS

JavaScript

Conceito Servidor

(voltar)

O conceito de servidor pode ser explicado como a parte que não aparece para o cliente

Na programação os profissionais que trabalham nesse lado(servidor) da TI são responsaveis por

uma parte diferente(Back-end) dos profissionais que trabalham na interface e interação do cliente(Front-end)

Eles estão procupados mais com o banco de dados do que com a interface qie um cliente visualiza

são responsaveis por criar sistemas, analizar dados,informações que muitas vezes aproximam a empresa dos clientes

criando assim uma individualização no atendimento por exemplo, ou uma acertividade maior em tendencias e lançamentos
